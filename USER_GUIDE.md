# Hướng Dẫn Sử Dụng Trên Điện Thoại & Chia Sẻ

Bạn có thể sử dụng ứng dụng ôn tập này trên điện thoại hoặc máy tính bảng một cách dễ dàng. Dưới đây là các cách để thực hiện:

## Cách 1: Gửi File Trực Tiếp (Đơn giản nhất)
Đây là cách nhanh nhất để chia sẻ mà không cần cài đặt gì thêm.

1.  **Tìm file**: Tìm file `standalone.html` trong thư mục dự án của bạn trên máy tính.
2.  **Gửi file**:
    *   Gửi file này qua **Zalo**, **Messenger**, **Telegram**, hoặc **Email** cho chính bạn hoặc bạn bè.
    *   *Lưu ý*: Nếu gửi qua Zalo/Messenger, hãy chọn chế độ "Gửi file" (đính kèm tệp) để tránh bị lỗi định dạng.
3.  **Mở trên điện thoại**:
    *   Trên điện thoại, nhấn vào file vừa nhận.
    *   Chọn **"Mở bằng Trình duyệt"** (Open with Browser) hoặc chọn Chrome/Safari.
    *   Ứng dụng sẽ chạy ngay lập tức!

## Cách 2: Sử Dụng Cloud (Google Drive / iCloud)
1.  Tải file `standalone.html` lên Google Drive hoặc iCloud Drive của bạn.
2.  Trên điện thoại, mở ứng dụng Drive/Files.
3.  Tìm file và mở nó. (Một số điện thoại có thể yêu cầu tải về máy trước khi mở bằng trình duyệt).

## Cách 3: Đưa Lên Web (Chuyên nghiệp hơn)
Nếu bạn muốn có một đường link (ví dụ: `ontap.github.io`) để ai cũng vào được mà không cần gửi file:

1.  Tạo một tài khoản **GitHub** (miễn phí).
2.  Tạo một kho chứa (repository) mới.
3.  Tải file `standalone.html` lên và đổi tên thành `index.html`.
4.  Vào **Settings** -> **Pages** -> Chọn nhánh **main** và nhấn **Save**.
5.  GitHub sẽ cung cấp cho bạn một đường link. Bạn chỉ cần gửi link này cho mọi người!

## Lưu Ý Quan Trọng
*   **Dữ liệu**: Tiến độ học (các câu đã thuộc) được lưu trên trình duyệt của từng máy. Nếu bạn học trên máy tính rồi chuyển sang điện thoại, bạn sẽ phải học lại từ đầu trên điện thoại (hoặc ngược lại).
*   **Giao diện**: Ứng dụng đã được tối ưu cho điện thoại, bạn có thể xoay ngang màn hình để có trải nghiệm tốt hơn nếu cần.
